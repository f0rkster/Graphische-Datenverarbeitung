Detailed Description

Defines the elements of a mesh.

Defines the elements of a mesh. A mesh alway has to consist of vertices and indices. All other attributes such as normals, colors, and texture coordinates are optional. If you do not want to support one of these attributes then mark it with a null pointer. If the mesh should not be textured then set the texture coordinates and the texture handle to null. Please consider that all attributes are passed per vertex, so the number of the normals, colors, and texture coordinates have to match the number of the vertices.
